## 工程新增功能描述

### 概述

`text-effects_a` 工程相比原始的 `text-effects` 工程，新增了多项功能增强，主要包括颜色定制、全屏预览、动态列表管理等功能。

---

### 新增功能清单

| 功能模块 | 新增/修改文件 | 功能描述 |
| :--- | :--- | :--- |
| 全屏预览页面 | `pages/FullScreen.ets` | 新增全屏展示页面，支持横屏模式 |
| 颜色选择器 | `pages/Index.ets` | 为三种文字效果添加颜色选择功能 |
| 自定义对话框 | `pages/Index.ets` | 添加自定义文字效果的对话框组件 |
| 动态列表管理 | `pages/Index.ets` | 支持动态添加和管理文字效果项 |
| 导航路由 | `pages/Index.ets` | 实现页面间导航跳转 |
| 视图组件增强 | `view/*.ets` | 三个视图组件添加颜色参数支持 |

### 详细功能说明

#### 1. 全屏预览功能

**文件：** `entry/src/main/ets/pages/FullScreen.ets`

**功能描述：**
- 新增全屏展示页面，支持将任意文字效果以全屏方式展示
- 自动切换横屏模式，提供更好的视觉体验
- 离开页面时自动恢复竖屏模式

**核心特性：**
```typescript
// 横屏切换逻辑
private changeOrientation(isLandscape: boolean) {
  let context = getContext(this) as common.UIAbilityContext;
  window.getLastWindow(context).then((lastWindow) => {
    let orientation = isLandscape ? window.Orientation.LANDSCAPE : window.Orientation.PORTRAIT;
    lastWindow.setPreferredOrientation(orientation, (err: BusinessError) => {});
  });
}
```

#### 2. 颜色定制功能

**文件：** `entry/src/main/ets/pages/Index.ets`

**功能描述：**
- 为歌词滚动、文字倒影、跑马灯三种效果添加颜色选择器
- 支持 5 种颜色：红色、黄色、蓝色、粉色、绿色
- 实时预览所选颜色效果

**支持的颜色选项：**
| 序号 | 颜色名称 | 对应颜色值 |
| :--- | :--- | :--- |
| 1 | 红色 | `Color.Red` |
| 2 | 黄色 | `Color.Yellow` |
| 3 | 蓝色 | `Color.Blue` |
| 4 | 粉色 | `Color.Pink` |
| 5 | 绿色 | `Color.Green` |

---

#### 3. 自定义文字效果添加

**文件：** `entry/src/main/ets/pages/Index.ets`

**功能描述：**
- 新增「新增」按钮，点击弹出自定义对话框
- 可选择效果类型（渐变、滚动、倒影、跑马灯）
- 可输入自定义文字内容（最多12字）
- 添加的效果自动加入动态列表

**对话框界面结构：**
```
┌─────────────────────────────┐
│ [选择效果下拉框]            │
├─────────────────────────────┤
│ [输入框：请输入内容]         │
├─────────────────────────────┤
│   [关闭]        [确认]      │
└─────────────────────────────┘
```

---

#### 4. 动态列表管理

**文件：** `entry/src/main/ets/pages/Index.ets`

**功能描述：**
- 使用 `arrText` 类管理效果数据
- 支持动态添加、删除、编辑效果项
- 列表项支持点击跳转到全屏预览
- 已添加的效果项支持颜色重新选择

**数据模型：**
```typescript
@Observed
export class arrText {
  effect: string;      // 效果类型名称
  content: ResourceStr; // 显示内容
  color: Color;        // 颜色配置
}
```

### 技术亮点

1. **响应式状态管理**：使用 `@State`、`@Link`、`@ObjectLink` 实现组件间状态同步
2. **自定义对话框**：使用 `CustomDialogController` 实现模态对话框
3. **路由导航**：使用 `NavPathStack` 实现页面间参数传递
4. **窗口管理**：使用 `window` API 实现横竖屏切换
5. **动态列表**：使用 `ForEach` 结合 `@Observed` 实现列表数据响应式更新
